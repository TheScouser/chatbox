Embedded Chat Widget Demo

Your Website Content

This is an example of how your chat widget will appear alongside your website content. The widget is embedded using an iframe and will maintain its functionality while being seamlessly integrated into your page.

Key Features:

  • Real-time AI responses
  • Knowledge-based answers
  • Responsive design
  • Easy integration

Visitors to your website can interact with the chat widget to get instant answers to their questions based on the knowledge you've provided to your AI agent.

How to Use This Test Page

  1. Copy your embed code from the dashboard
  2. Replace the iframe code above (line 67) with your actual embed code
  3. Open this HTML file in your browser to test
  4. Verify that the chat widget loads and functions correctly

Example Embed Code:

<iframe src="https://your-domain.com/embed/your-agent-id" width="400" height="600" frameborder="0"></iframe>
